# Lab #1: Statistical Analysis of Census Data
Objective: Explore demographic and income distributions using Python.

## 🛡️ Защита работы (Как рассказывать преподавателю)

1.  **Цель работы:** "В данной работе я провел первичный исследовательский анализ (EDA) набора данных переписи населения (Census Income). Основная задача — изучить распределение демографических признаков и их связь с уровнем дохода."
2.  **Что сделано (Техническая часть):**
    *   "Использовал библиотеку **Pandas** для загрузки и предобработки данных."
    *   "Рассчитал основные статистические метрики: среднее, медиану и моду для возраста и рабочего времени."
    *   "С помощью метода `pivot_table` (сводные таблицы) проанализировал зависимость возраста от уровня образования и пола."
3.  **Результаты:** "Выявлено, что уровень образования напрямую коррелирует с более высокой медианной зарплатой. Для визуализации распределения уровней образования я построил столбчатую диаграмму в **Matplotlib**."
4.  **Ответ на вопрос «А как это работает?»:** "Скрипт автоматически фильтрует данные, удаляет лишние пробелы в категориальных переменных и генерирует наглядный график распределения."
