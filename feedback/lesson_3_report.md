## 📋 Разбор ошибок:
Сэр, логика условий построена хорошо, но есть две "технические" ошибки, которые не дадут программе запуститься:

1. **Точка с запятой (Semicolon):** Вы забыли поставить `;` в конце команд внутри блоков `else` и второго `if`. В Java это обязательный символ после каждой операции вывода.
   * *Ошибка:* `System.out.println("...")`
   * *Исправление:* `System.out.println("...");`

2. **Структура if-else if:** Ваша логика работает, но профессиональнее использовать связку `if` -> `else if` -> `else`. Это гарантирует, что выполнится только один блок кода.
