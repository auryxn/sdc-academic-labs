## 📂 Структура репозитория
- `lessons/` — теория и задания от меня.
- `homework/` — папка для вашего кода.
- `feedback/` — мои отчеты о проверке вашего кода.

## 🚀 Как мы работаем
1. Я создаю файл с уроком в `lessons/`.
2. Вы создаете решение в `homework/` (например, `homework/Lesson_1/Main.java`).
3. Вы делаете **Git Push** из своей IDE или консоли.
4. Я вижу обновление, проверяю код, пишу фидбек и ставлю оценку в `feedback/`.
5. Если всё ок — открываю следующий урок.
