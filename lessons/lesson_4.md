## 1. Цикл for
Используется, когда мы заранее знаем, сколько раз нужно выполнить код.
\`\`\`java
for (int i = 0; i < 5; i++) {
    System.out.println("Итерация: " + i);
}
\`\`\`
- \`int i = 0\` — начальная точка.
- \`i < 5\` — условие (пока оно верно, цикл работает).
- \`i++\` — шаг (увеличиваем счетчик после каждого круга).

## 2. Цикл while
Используется, когда мы не знаем точного количества повторений, а ждем выполнения условия.
\`\`\`java
while (isWorking) {
    // Делаем что-то...
}
\`\`\`
