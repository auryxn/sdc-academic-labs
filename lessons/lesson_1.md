# 🎓 Java Course: Lesson 1 - The Beginning

## 1. Что такое Java?
Java — это строго типизированный объектно-ориентированный язык. Главная фишка: **"Write Once, Run Anywhere"** (Напиши один раз, запускай где угодно). Это возможно благодаря JVM (Java Virtual Machine).

## 2. Структура программы
В Java всё находится внутри **классов**. 
- Имя файла должно совпадать с именем публичного класса.
- Точка входа в любую программу — метод `public static void main(String[] args)`.

## 3. Переменные и Типы данных
В Java мы обязаны говорить компьютеру, какой тип данных мы храним:
- `int` — целые числа (например, 10).
- `double` — дробные числа (например, 3.14).
- `boolean` — правда или ложь (`true`/`false`).
- `String` — текст (обязательно в двойных кавычках: "Hello").

## 🛠 Задание №1: "Первый Контакт"
1. Создайте файл `Main.java`.
2. Напишите в нем программу, которая:
   - Создает переменную типа `int` с вашим возрастом.
   - Создает переменную типа `String` с вашим именем.
   - Выводит в консоль фразу: "Привет, меня зовут [Имя], мне [Возраст] лет".
