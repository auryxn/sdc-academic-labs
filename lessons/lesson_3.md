# 🎓 Java Lesson 3: Условные операторы и Логика

Сэр, программа становится умной, когда она может выбирать путь.

## 1. Оператор if-else
Синтаксис:
\`\`\`java
if (условие) {
    // Код, если условие ИСТИННО (true)
} else {
    // Код, если условие ЛОЖНО (false)
}
\`\`\`

## 2. Операторы сравнения
- \`==\` равно
- \`!=\` не равно
- \`>\` больше / \`<\` меньше
- \`>=\` больше или равно / \`<=\` меньше или равно

## 3. Логические операторы
- \`&&\` (И) — оба условия должны быть верны.
- \`||\` (ИЛИ) — хотя бы одно условие верно.
