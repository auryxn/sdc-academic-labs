# Lab #2: Visual Data Analysis of Cardiovascular Disease
Objective: Analyze medical indicators and their correlation with CVD.

## 🛡️ Защита работы (Как рассказывать преподавателю)

1.  **Цель работы:** "Исследование факторов риска сердечно-сосудистых заболеваний (CVD) с применением продвинутых инструментов визуализации в Python."
2.  **Что сделано (Техническая часть):**
    *   **Препроцессинг:** "Преобразовал возраст из дней в полные годы и рассчитал индекс массы тела (BMI) по формуле (вес / рост^2)."
    *   **Визуализация:** "Использовал библиотеку **Seaborn** для построения гистограмм распределения веса и возраста."
    *   **Анализ выбросов:** "Построил диаграммы размаха (**Boxplots**) и рассчитал межквартильный размах (IQR) для выявления аномалий в данных (экстремально низкий/высокий рост и вес)."
3.  **Гипотеза:** "Проверил клиническую гипотезу: в группе курящих мужчин 60-65 лет с высоким давлением риск заболевания в несколько раз выше, чем в контрольной группе."
4.  **Результаты:** "Графики четко показывают, что вероятность CVD значительно возрастает после 50 лет, что коррелирует с теорией."
